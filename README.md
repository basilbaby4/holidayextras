# holidayextras
 
 Build an API

 ## Installation:

```sh
$ git clone <repository-url>

```
Install dependencies

```sh
$ npm install
```

Configure Database

```sh
Create mysql database and update config/config.json [development] file. 
No Need to create tables.
```
Start the server:

```sh
$ npm start
```
View the website at: http://localhost:3000
